## 背景

`boot` 此工程为测试一些中间件的使用已经一些中间件版本的兼容性。

这个仓库的目标是：

1. 学习中间件的使用。
2. 学习搭建脚手架工程。

## 安装

这个项目使用 jdk11 和 maven3。请确保你本地安装了它们。


## 使用说明

> 启动BootApplication.java
